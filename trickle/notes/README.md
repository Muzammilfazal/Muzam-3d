# Muzam 3D Editor

Muzam 3D is a modern web-based 3D animation and modeling application running entirely in the browser.

## Features
- **Docked UI Layout**: Clean side-panel traditional editor docking.
- Three.js powered realtime rendering
- Basic primitives support (Cube, Sphere, Plane, Cylinder)
- Recursive hierarchy and selection state management
- Keyframe Animation & Rendering support

## Architecture
- React 18 (CDN)
- Tailwind CSS (Dark Glass Theme)
- Three.js via WebGL for 3D Viewport
- React Context API for application state
